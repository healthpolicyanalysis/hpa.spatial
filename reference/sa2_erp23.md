# Estimated Residential Population 2023

(Estimated) population data at the SA2 level (2021 edition) from the
ABS.

## Usage

``` r
sa2_erp23
```

## Format

\`sa2_erp23\` A data frame with 2,472 rows and 2 columns:

- sa2_code:

  SA2 code (2021 edition)

- erp:

  estimated number of residents

## Source

\<https://www.abs.gov.au/statistics/people/population/regional-population/latest-release#data-downloads\>
