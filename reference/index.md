# Package index

## All functions

- [`check_for_internal_polygon()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/check_for_internal_polygon.md)
  :

  Check for (and return if so) the shape if it is "internal" (contained
  within `hpa.spatial`/`hpa.spatial.data`)

- [`create_child_geo()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/create_child_geo.md)
  : Create a child geography (polygon) which fits neatly into a parent
  geography

- [`get_correspondence_tbl()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/get_correspondence_tbl.md)
  :

  Get a correspondence table, either via
  [`strayr::read_correspondence_tbl`](https://runapp-aus.github.io/strayr/reference/read_correspondence_tbl.html)
  or by creating one with `make_correspondence_tbl`

- [`get_mb21_poly()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/get_mb21_poly.md)
  : Get Mesh Blocks (2021 edition) polygons

- [`get_mb21_pop()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/get_mb21_pop.md)
  : Get Mesh Blocks (2021 edition) and population counts

- [`get_mmm19_poly()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/get_mmm19_poly.md)
  : Modified Monash Model polygons (SA1, 2016 edition, polygons with
  MMM, 2019 edition column)

- [`get_polygon()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/get_polygon.md)
  :

  Get shapefiles from ABS (via `strayr` and `absmapsdata` or via
  `hpa.spatial` and `hpa.spatial.data`)

- [`make_correspondence_tbl()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/make_correspondence_tbl.md)
  : Create a correspondence table between any two geographies

- [`map_data_with_correspondence()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/map_data_with_correspondence.md)
  : Map data between editions using correspondence tables from the ABS.

- [`read_hpa_spatial_data()`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/read_hpa_spatial_data.md)
  : Read a dataset from
  \<github.com/healthpolicyanalysis/hpa.spatial.data/\>

- [`sa2_erp23`](https://healthpolicyanalysis.github.io/hpa.spatial/reference/sa2_erp23.md)
  : Estimated Residential Population 2023
